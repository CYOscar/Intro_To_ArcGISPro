Step 1: Setting Up ArcGIS Pro

  1. Open ArcGIS Pro that is already installed on the computer.
  2. Click on the ArcGIS organization's URL. type 'nau', click enter and sign in with your student credentials.
    Note: If you would like to have your own AcrGIS Pro software on your computer, you can get a subscription from Esri.
  3. Once you are signed in, click map for a new project.
  4. The New Project window appears. By default, projects are saved in a new folder. Click Map and create a name for the project. Find your folder where your project will be stored.
  5. Click OK.
  6. On either side of the map are panes. By default, the Contents and Catalog panes are open, although other panes may be open if you've used ArcGIS Pro before. The Contents pane lists the layers on the map and the Catalog pane lists all files associated with the project.

Step 2: Add Basemap and Data

  1. On the Map tab, click Basemap. Pick a basemap of your choice.
  2. To add data, on the Map tab, in the Layer group, click the Add Data button to find your folder.
  3. After adding data, the layers are listed on the Content pane.
  4. If the standalone table is .xls, .xlsx or csv from Excel, the XY table will need to be converted to point layer to be visible on the map. First, go to Map tab on top and click on XY Table To Point under Layer.
  5. Once you click on XY Table To Point, you should see a pop-up page on the right. Add the table to Input Table. Create a name for this layer on Output Feature Class. Make sure the X field and Y field are correct and the coordinate system is selected for the UTM zone. Then, click run.
  6. Now, you can see points on the map. If you want to
    change the symbol of each layer, click the symbol. A Symbology pane will appear on the right to edit.
  7. To edit features in a layer, click the layer name to highlight and edit, click Edit tab and click Snapping to activate. Then, click Create.
  8. You should see a pop-up on the right for Create Features. You can now add feature on the map.
  9. Once you add a feature on the map, double-click the feature to confirm. If you need to delete a feature, make sure to click the feature and click delete that is next to Create and Modify. You can also click discard to delete all edits.
  10. After making changes to the features, click save to accept changes.
  11. Click save on top-right corner to save your project.

Step 3: Create Map Layout

  1. To create a map layout, click on the Insert tab and click New Layout.
  2. Pick a map layout size. The most common size is 8.5" x 11" Letter.
  3. The map layout is added to the map view as a new tab. In the Catalog pane, there is now a folder called Layouts. Expand Layouts. In the Layout folder, right-click Layout and choose Rename to give it an appropriate name.
  4. Save your project.
  5. Your layout is now ready to add elements, including a map frame, legend, north arrow, scale bar and title. On the Insert tab, click Map Frame to add a map frame to the layout.
  6. Once you click a map from the Map frame, draw a box inside the boundaries of the layout. Adjust the map size after it has been created.
  7. If you want to move the map within the map frame, right-click the map and click Activate.
  8. In the map frame, pan the map.
  9. After adjustments are made, go to Layout tab and click Close Activation.
  10. Save your project.
  11. Next, to add a legend, click Legend on the Insert tab. Pick a legend.
  12. Then, add the North Arrow, Scale Bar, and Text for the title.
  13. To change the appearance of each element, double-click the element on the map layout and Element page will appear on the right.
  14. After incorporating all the elements into the map layout, your final layout should resemble something like this.
  15. Now, we can share the map layout. Go to Share tab and click Export Layout. A pop-up window will appear on the right.
  16. Choose the right File Type and choose the folder where the layout will be saved.
    Note: The default file type is PDF. If the map is a raster image, it is recommended to save as PNG.
  17. Accept the other default settings. Click Export.
