Create a File Geodatabase

First method: Use the Cataolog pane in ArcGIS Pro

  1. Start ArcGIS Pro and open the Catalog pane, if necessary.
  2. Right-click Databases or a folder under Folders in the Catalog pane and click New File Geodatabase.
  3. On the New File Geodatabase box, browse to the location where you want to create a file geodatabase, type a name, and Click Save.

    4. Second method: Run the Create File Geodatabase tool

  4. Go to the Analysis tab and in the Tools pane, type Create File Geodatabase in the search box.
  5. Specify the folder location where you want the file geodatabase to be created, and type a name for the file geodatabase.
  6. Click Run.
